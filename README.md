# MetaTrack

Desenvolvido em Python, este projeto tem como objetivo explorar o conceito de programação multiparadigma, unindo abordagens imperativas, funcionais e orientadas a objetos. A aplicação calcula quanto o estudante precisa obter na próxima avaliação para atingir a média mínima para aprovação, com base nas notas já obtidas. Além de útil, o projeto serve como prática para compreender como diferentes paradigmas podem coexistir e se complementar em uma mesma solução.


Project Description:
Developed in Python, this project aims to explore the concept of multiparadigm programming, combining imperative, functional, and object-oriented approaches. The application calculates how much a student needs to score on the next exam to reach the minimum average required for approval, based on the grades already obtained. In addition to being useful, this project serves as a practical exercise to understand how different programming paradigms can coexist and complement each other within the same solution.
